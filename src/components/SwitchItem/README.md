### SwitchItem 组件

| 参数  | 说明         | 类型      | 默认值  |
| :---- | :----------- | :-------- | :------ |
| value | 开关选中状态 | _boolean_ | `false` |

| 事件名 | 说明               | 回调参数         |
| :----- | :----------------- | :--------------- |
| input  | 开关状态切换时触发 | _value: boolean_ |

- 你也可以使用 v-model 代替 value 和 input
